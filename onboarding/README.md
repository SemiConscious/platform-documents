## Quick Start by Role

### Software Developer

1. Complete [Developer Onboarding](./developer.md)
2. Read [Global Architecture](/architecture/global-architecture.md)
3. Review [Repository Inventory](/services/repository-inventory.md)
4. Start with assigned tasks

### Platform Engineer

1. Complete [Platform Engineer Onboarding](./platform-engineer.md)
2. Read all [Runbooks](/operations/runbooks/)
3. Shadow on-call for 2 weeks
4. Complete scenario training

### Salesforce Developer

1. Complete [Developer Onboarding](./developer.md)
2. Focus on Salesforce repositories
3. Set up Salesforce development environment
4. Review AVS package architecture
